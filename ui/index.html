Site sur les Video Games

Ce Site web présente les différentes vidéo games qui existes et le détail de chaque video games développée (Prix , PLateform...). la source de ces données est sur le site : Data Games: Durant ce projet nous avons utilisé :
  • Docker :
    • Le module Mongodb pour la création de la bases de données a partir d'un fichier JSON
    • Le module Appolo-server pour la création du serveur et pour gérer les différentes routes
    • Le module Graphql pour lancer les requettes et récuperer les données de notre base de données mongo
  • D3.js pour la visualisation de données générée
  • Bootstrap et CSS pour le style des pages html
